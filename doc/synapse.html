get_live_traces/2

get_live_traces(EventSource::fun(() -> event()), TraceEnd::fun((event()) -> false | include | exclude)) -> ok | {error, string()}

Register a trace receiver server that will direct new {trace, Trace} messages to the calling process as traces are detected. This is used for tracing continous systems, such as network traffic. The trace server is spawned with the specified event source function and trace end identifier function. It collates events until TraceEnd identifies the end of a trace, then the calling process is informed and it continues monitoring the event source.
